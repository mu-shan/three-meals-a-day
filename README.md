# 今天吃点啥

给一家三口使用的一日三餐随机菜单。打开页面即可生成早餐、午餐和晚餐，支持整餐重抽、单道换菜和自动汇总今日采购清单。

## 家庭规则

- 一家三口共用一桌菜。
- 米饭午餐至少 3 道菜。
- 米饭午餐必含宝宝固定菜：西红柿炒鸡蛋、清炒土豆丝、清炒豆芽三选一。
- 米饭午餐必含至少一道妈妈辣菜。
- 爸爸不设忌口。
- 晚餐至少一道宝宝友好菜，整体口味更清淡。
- 标记“喜欢”的菜会以普通菜 2 倍的权重参与随机。
- 标记“不喜欢”的菜会立即替换，之后不再进入随机菜单。

## 口味偏好

每道菜都可以直接标记“喜欢”或“不喜欢”。偏好会根据菜品标签自动归入宝宝、妈妈或全家；页面顶部的“口味偏好”可以集中查看、取消喜欢，或者把不喜欢的菜重新加入菜单。

## 本地运行

```bash
pnpm install
pnpm dev
```

## 验证

```bash
pnpm test
pnpm typecheck
```

## 技术与素材

- 页面使用 Vue 3、TypeScript、Vite 和 `animal-island-vue`。
- 菜品与家庭插画来自 [Twemoji](https://github.com/twitter/twemoji)，通过 jsDelivr 加载，图形采用 CC-BY 4.0 授权。
- 图片不可用时会自动切换到项目内置的餐盘 SVG 占位图。
- `animal-island-vue` 仓库声明 MIT 协议，同时 README 提示仅限学习和非商业使用；本项目按家庭自用场景实现。

## 数据保存

当天菜单和家庭口味偏好保存在浏览器 `localStorage` 中。刷新后继续显示同一天的选择，喜欢与不喜欢记录也会长期生效；日期变化后会生成新菜单。浏览器禁止本地存储时，随机功能仍可在当前页面正常使用。
